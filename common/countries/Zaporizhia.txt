# Нарратив: Запорозька Січ як напівнезалежна козацька республіка на Нижньому Дніпрі.
# Витоки у XV ст. як прикордонні громади; розквіт у XVI–XVII ст.; провідна роль у
# Визвольній війні 1648–1657 рр.; складні протекції між Річчю Посполитою, Московією
# та Османсько‑Кримським світом; руйнація Січі 1775 р., частина козацтва переходить
# до Чорноморського/Кубанського війська. У XIX ст. – інтеграція імперськими інститутами.

# Історичні механіки (ідеї для геймплею):
# - Січ та її релокації: Томаківська → Базавлуцька → Микитинська → Чортомлицька →
#   Олешківська (під Османами) → Нова Січ. Рішення на перенесення столиці з кулдауном,
#   вимоги: степ/луки, сусідство з річками, відсутність чужого форту; модифікатор "столиця Січі".
# - Чайки/морські рейди: події/рішення для рейдів узбережжя Чорного й Азовського морів,
#   здобич, морський професіоналізм, погіршення відносин з Османами/Кримом
#   (напр. ottoman_chaiky_humiliation).
# - Козацький реєстр: розширити/скоротити реєстр (вплив на утримання армії, лояльність
#   станів, відносини з сюзеренами РП/Московії).
# - Демократія Січі: виборність кошового отамана (подієвий цикл із впливом старшини),
#   бонуси в степах/луках, фронтирні слободи на прикордонних провінціях.
# - Ключові рубежі: Кодацька фортеця як виклик/ціль подій; інтеграція з Лівобережжям.
