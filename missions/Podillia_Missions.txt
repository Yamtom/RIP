# =========================================================================
# PODILLIA DEVELOPMENT MISSIONS - TECHNICAL INTEGRATION LAYER
# =========================================================================
# HISTORICAL CONTEXT (1444-1648):
# Podillia represents the strategic borderland between the Polish-Lithuanian
# Commonwealth and the Ottoman/Crimean sphere of influence. Unlike Volhynia's
# more centralized path or Galicia's magnate-driven development, Podillia
# follows a trajectory analogous to:
#
# 1. CROATIAN MODEL (from croatian_missions.txt):
#    - Small principality facing larger imperial powers
#    - Naval/Fortress-based defense strategy
#    - Resistance to foreign domination (Ottomans vs. Ottomans)
#    - Eventual accommodation with neighboring powers
#    - Focus on border defense and local autonomy
#
# 2. MAZOVIAN MODEL (Hypothetical, based on historical reality):
#    - Semi-independent duchy within larger commonwealth
#    - Delayed incorporation into unified state
#    - Steppe/nomadic border management (vs. Croatian maritime)
#    - Magnate-led territorial expansion
#    - Balance between Orthodox and Catholic traditions
#
# PODILLIA'S UNIQUE PATH (1444-onwards):
# - 1444-1569: Independent Voivodeship under Polish suzerainty
# - 1569-1648: Integrated into Polish-Lithuanian Commonwealth
# - Border wars with Ottomans, Crimea, and Cossacks
# - Economic development through trade and agriculture
# - Religious diversity (Orthodox, Catholic, Jewish communities)
# - Fortress cities: Kamianets-Podilskyi, Bar, Khotyn
#
# ========== GAME MECHANICS INTEGRATION ==========
# This mission tree implements PDL three strategic paths with technical references:
#
# Path A: "Carpathian Bastion" (Slot 1, 5 missions)
#   - Unlocks: pdl_carpathian_bastion_reform (RIP_government_reforms.txt)
#   - Government Names: pdl_carpathian_bastion trigger block (000_RIP_names.txt)
#   - Synergies: Volhynia vln_voivode_council (shared military traditions)
#   - Mechanics: Garrison culture, fort maintenance bonuses, defensive bonuses
#
# Path B: "Frontier Republic" (Slot 2, 4 missions)
#   - Unlocks: pdl_frontier_republic_reform (RIP_government_reforms.txt)
#   - Government Names: pdl_frontier_republic trigger block (000_RIP_names.txt)
#   - Synergies: Volhynia vln_cossack_host_reform (cavalry/estate integration)
#   - Mechanics: Cossacks_mechanic government ability, cavalry bonuses, steppe control
#   - Estate Integration: estate_cossacks with influence mechanics
#
# Path C: "Magnate Dominion" (Slot 3, 5 missions)
#   - Unlocks: pdl_magnate_dominion_reform (RIP_government_reforms.txt)
#   - Government Names: pdl_magnate_dominion trigger block (000_RIP_names.txt)
#   - Synergies: Galicia hlc_crown_and_sejm_reform (noble estates), Mazovia analog
#   - Mechanics: Trade power, noble influence, governing capacity expansion
#   - Estate Integration: estate_nobility with autonomy mechanics
#
# Unifying Path: "Universal Supremacy" (Slot 4, consolidation)
#   - Unlocks: pdl_grand_podillia_reform (final government tier)
#   - Requires: Completion of primary path + territorial control
#   - Synergy Missions: PDL_galician_compact (HLC integration), PDL_trade_corridor
#
# Optional Path: "Religious Tolerance" (Slot 5, cultural overlay)
#   - Unlocks: pdl_religious_tolerance_reform (optional overlay reform)
#   - Compatible: Can combine with any primary path
#   - Mechanics: Religious unity, missionary strength, tolerance mechanics
#   - Religious Focus: Orthodox/Catholic/Jewish coexistence (Podillian uniqueness)
#
# ========== MODIFIER SYSTEM REFERENCE ==========
# Modifiers created by these missions (must be declared in RIP_government_reforms.txt):
# - pdl_khotyn_fortress (fortress_reinforcement+garrison bonuses)
# - pdl_garrison_tradition (garrison culture development)
# - pdl_ottoman_resistance (defensive standing modifiers)
# - pdl_steppe_masters (cavalry bonuses, Cossack integration)
# - pdl_host_assembly (estate_cossacks influence)
# - pdl_secure_crimea_raids (cavalry tradition from steppe raiding)
# - pdl_magnate_authority (noble influence expansion)
# - pdl_develop_trade (merchant availability and trade efficiency)
# - pdl_agricultural_prosperity (production bonus from steppe agriculture)
# - pdl_toll_roads (trade steering and merchant Republic bonuses)
# - pdl_magnate_sejm (autonomous sejm mechanics, max_absolutism cap)
# - pdl_grand_dominion (prestige and diplomatic reputation)
# - pdl_jewish_communities (tolerance for jewish, trade efficiency)
# - pdl_confessional_equilibrium (religious unity, stability bonuses)
# - pdl_cultural_renaissance (development cost reduction, institution spread)
#
# ========== CROSS-REFERENCE MAPPING ==========
# Synergy Missions (unlock when related regions/tags achieve specific state):

# ==================================================
# SYNERGY MISSIONS: Integration with Galicia & Volhynia
# ==================================================
# Cross-Regional Integration Framework:
# These missions create strategic interconnections between PDL, HLC (Galicia),
# and VLN (Volhynia), reflecting the historical interactions of the Eastern
# Ruthenian regions during the Polish-Lithuanian Commonwealth period.
#
# TECHNICAL SYNERGY MECHANICS:
# 1. PDL + HLC (Galicia) Integration:
#    - Trigger: PDL_galician_compact mission (below)
#    - Requirement: Alliance + HLC independence
#    - Mechanical Effects: 
#      * Shared trade power in Red Ruthenia area
#      * Improved relations due to religious/cultural similarity
#      * Cross-reference with hlc_crown_and_sejm_reform (noble cooperation)
#      * Flag: pdl_galician_compact enables mission PDL_trade_corridor
#
# 2. PDL + VLN (Volhynia) Integration:
#    - Trigger: PDL_trade_corridor mission (below)
#    - Requirement: VLN independence, border proximity, diplomatic ties
#    - Mechanical Effects:
#      * Enhanced trade efficiency in shared provinces/areas
#      * Cavalry tradition bonus (both have Cossack paths)
#      * Religious unity boost (both Orthodox-Catholic balance)
#      * Cross-reference with vln_voivodeship_reform and vln_cossack_host_reform
#
# 3. PDL Path Interactions with Government Reforms:
#    - Carpathian Path (Fortress) unlocks: pdl_carpathian_bastion_reform
#      triggers government_names/pdl_carpathian_bastion block (000_RIP_names.txt)
#      → Government titles: FORT_CAPTAIN → GARRISON_COMMANDER → BASTION_VOIVODE
#
#    - Frontier Republic (Cossack) unlocks: pdl_frontier_republic_reform
#      triggers government_names/pdl_frontier_republic block
#      → Government titles: HOST_VOIVODE → HETMAN_VOIVODE → HOST_COMMANDER
#      → Enables government_abilities = { cossacks_mechanic }
#
#    - Magnate Dominion (Noble) unlocks: pdl_magnate_dominion_reform
#      triggers government_names/pdl_magnate_dominion block
#      → Government titles: MAGNATE_DUKE → MAGNATE_KING → AUTONOMOUS_VOIVODE
#      → Enhanced estate_nobility mechanics and autonomy interaction
#
# MODIFIER STACKING CONTEXT:
# Multiple path-specific modifiers are designed to NOT conflict:
# - pdl_khotyn_fortress (Carpathian) = garrison + fort bonuses
# - pdl_steppe_masters (Frontier) = cavalry + nomad interaction bonuses
# - pdl_magnate_rule (Dominion) = trade + noble influence bonuses
# - pdl_tolerance_path (Optional) = religious bonuses (stacks with any path)
#
# DIPLOMATIC SYSTEM INTEGRATION:
# Synergy missions create opinion modifiers that increase likelihood of:
# - Coalition formation against common Ottoman/Crimean threats
# - Trade league membership (when available)
# - Military access grants
# - Shared stability and religious unity benefits
#
# EVENT SYSTEM HOOKS (for future event designers):
# These missions create flags usable in decision/event system:
# - has_country_flag = pdl_galician_compact
# - has_country_flag = pdl_trade_corridor
# - has_country_flag = pdl_ottoman_raids_survive  (from Carpathian path)
# - has_country_flag = pdl_cossack_recruited      (from Frontier path)
# - has_country_flag = pdl_magnate_path           (from Magnate path)
