# === ІСТОРИЧНИЙ КОНТЕКСТ (XV–XVIII ст.) ЧЕРНІГІВЩИНА ===
# НАРРАТИВ: Чернігівщина — прикордонний лісостеповий регіон між Литвою/РП та Московією;
# осердя Лівобережної Гетьманщини з полковим центром у Чернігові (Чернігівський полк).
# Ключові міста: Батурин (резиденція І. Мазепи), Ніжин (культурний центр), Глухів (після 1708).
# Пік політичної ваги у добу Руїни (1657–1687) та за гетьманства І. Мазепи (1687–1709);
# 1708 р. — знищення Батурина царськими військами в розплату за швецько-козацький союз;
# XVIII ст. — поступова інтеграція до російської адміністрації, секуляризація монастирів,
# утворення чернігівської губернії (1802).
#
# КЛЮЧОВІ ПОСТАТІ ЧЕРНІГІВЩИНИ:
# - Іван Мазепа (1639–1709): гетьман, найбільший покровитель Чернігівщини; розширив 
#   Батурин, заснував видавництва й архітектурні проекти (Чернігівський колегіум)
# - Козак Мазепа: спершу проруський率, потім спроба союзу зі Швецією (1708), остаточна 
#   перемога Петра I при Полтаві (1709)
# - Старшинські роди Чернігівщини: Лизогуби, Апостоли, Розумовські — впливові генерали
#   й радники; конкуренти гетьманів, іноді скинувалися з влади, іноді підносилися
# - Семен Розумовський (1701–1771): син козака, став російським фельдмаршалом; репрезентує
#   поступову російськизацію чернігівської старшини
#
# ІСТОРИЧНІ МЕХАНІКИ (ідеї для геймплею):
# - Форпост Лівобережжя: бонуси оборони й постачання в лісостепу, подієння розбудови
#   фортець (Батурин, Глухів), слободи на кордонах як буферна зона проти татар.
# - Старшинські роди: династичні лінії (Лизогуби, Апостоли, Розумовські тощо) як 
#   впливові генерали/радники; інтриги довкола гетьманських виборів; можливість 
#   підвищення їхньої влади або супресії залежно від гетьманської політики.
# - Православні монастирі: культурно-освітні центри (Чернігівський колегіум, монастирі);
#   взаємодії з Церквою (легітимність, стабільність, миротворчість, місіонерство);
#   секуляризація XVIII ст. як звернення у ользу держави.
# - Московська протекція: угоди, що дають безпеку й модернізацію, але обмежують автономію;
#   подієння Колегії (російське управління через губернаторів), стандартизація адміністрації,
#   російське гарнізування, інтеграція в російські військові структури.
# - Батурин як центр: як резиденція гетьмана, важливий форпост; його знищення 1708 р.
#   як символічна подія кінця гетьманської незалежності.
#
# ІСТОРИЧНІ РЕФЕРЕНСИ:
