# === ІСТОРИЧНИЙ КОНТЕКСТ (XVII–XVIII ст.) ===
# НАРРАТИВ: Гетьманщина як козацька держава, що постала з повстання Богдана Хмельницького
# (1648) і балансувала між Річчю Посполитою, Московією та Османською Портою. 
# Переяславські домовленості 1654 р. — спроба гарантувати автономію; період Руїни (міжусобиці,
# правобережно-лівобережний поділ 1657–1687); І. Мазепа й його шведський союз (1708–1709);
# обмеження автономії через Малоросійську колегію; остаточна ліквідація інститутів автономії
# у 1760–1780-х роках царством Катерини II.
#
# КЛЮЧОВІ ПОСТАТІ (XVII–XVIII ст.):
# - Богдан Хмельницький (1595–1657): засновник Гетьманщини, повстанець проти польської панщини
# - Юрій Хмельницький (1641–1682): син Богдана, слабкий правитель, під впливом старшини
# - Іван Виговський (1609–1664): правобережний суперник, спроба Гадяцької унії з Польщею
# - Петро Дорошенко (1627–1698): правобережний магнат, прагматично переходив від Польщі до Туреччини
# - Іван Самойлович (1633–1690): лівобережний гетьман, проруськи орієнтований, перемагав супротивників
# - Іван Мазепа (1639–1709): найвеличніший гетьман, централізатор, модернізатор культури й армії;
#   спроба втекти до Швеції (1708) і поразка при Полтаві (1709)
#
# ІСТОРИЧНІ МЕХАНІКИ (ідеї для геймплею):
# - Генеральна (козацька) рада й вибори гетьмана: вибір системи ("демократична" vs "централізована"),
#   історичні подієва конкуренція старшинських кланів, іноземний вплив (легітимність/корупція).
# - Пакти й статті: вибір у найважливіших моменти між союзними моделями:
#   * Переяславські договори (1654) — російська протекція, обмеження автономії
#   * Гадяцький трактат (1658) — союз з Польщею, розширення прав Гетьманщини
#   * Коломацькі статті (1687) — московські умови, поступова централізація
# - Полково-сотенний устрій: адміністративна структура козацтва; слободи на кордонах;
#   православні монастирі як центри культури й легітимності.
# - Правобережний/Лівобережний вибір: важливі подієння про лояльність регіонів, ризик інтервенцій
#   від РП / Московії / Османів; кризи престолонаступництва гетьмана (див. hetman_succession_crisis).
# - Реформи Мазепи й Розумовського: модернізація армії, заснування університетів (Могилянська
#   академія), покровительство культурі (архітектура, друкарство) — в обмін на централізацію
#   й конфліктність з Петербургом та утиском демократичних традицій Радой.
# - Економіка Гетьманщини: торгівля солі, хліба; цехові гільдії; експорт в Голландію, Англію;
#   експлуатація селів через панщину; взаємодія зі станами Церква/Козаки/Міщане/Шляхта.
#
# ІСТОРИЧНІ РЕФЕРЕНСИ:
